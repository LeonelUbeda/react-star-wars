## React app using SWAPI (Star Wars API)

Vercel deploy link: https://react-star-wars-api.vercel.app/

Api link: https://swapi.dev/

## About

Code challenge completed in 8 hours

## Libraries used:

- tailwind
- classnames
- Jotai (for cache)

Api responses are saved to localStorage
